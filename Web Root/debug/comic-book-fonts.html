Comic Book Font Test

Based on terrific feedback from Alen Yen, Steve Schmill, and Matt Young. These are the best free webfonts I can find so far, all from Font Squirrel. I'm glad they're free, but not thrilled with any of them. My questions to you:

  • Are there better free web fonts that look like hand-lettered comics?
  • If not, what's the best of this bunch?

Thanks in advance!

UPDATE: I just bought the webfont of Spinner Rack from Blambot for $25. It doesn't have a non-italic bold, but I wasn't going to use those anyway. The bold italic looks gigantic here. Next step is using it on real content!
